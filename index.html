🔍 BigQuery Analytics Interface

Run pre-defined analytics queries against the FlowSphere dataset. Queries execute via Google Cloud Functions connected to BigQuery.

SELECT zone_name, AVG(density_score) as avg_density,
       MAX(density_score) as peak_density,
       COUNT(CASE WHEN density_score > 5.5 THEN 1 END) as critical_events
FROM `flowsphere.crowd_analytics.zone_density`
WHERE event_date = CURRENT_DATE()
GROUP BY zone_name
ORDER BY avg_density DESC;

Select a query and click Execute
